Análise de Mercado de Ações

Projeto desenvolvido com planilhas para análise de operações no mercado de ações.

Objetivo

Realizar a análise de operações de compra e venda no mercado de ações, utilizando tratamento de dados, estatísticas, filtros, normalização e previsão temporal.

Etapas realizadas

* Criação da cópia do arquivo `manipulacao_mercado_acoes`
* Renomeação do arquivo para `analise_mercado_acoes`
* Normalização da coluna `preco`
* Criação de filtros para operações de compra e venda
* Consulta de nome e e-mail do cliente pelo código
* Cálculo de variância e desvio padrão do preço
* Criação da aba `tendencia_temporal`
* Contagem de operações por mês em 2023
* Previsão da quantidade de operações do mês 12
* Cálculo do erro entre previsão e valor real

Ferramentas utilizadas

* Microsoft Excel ou Google Sheets
* GitHub
